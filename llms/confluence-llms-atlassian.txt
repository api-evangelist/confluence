# Atlassian

> Atlassian is a global enterprise software company that builds tools for software development, project management, collaboration, and IT service management. Founded in 2002 in Sydney, Australia, Atlassian serves over 300,000 customers worldwide — from startups to Fortune 500 enterprises — helping teams plan, track, collaborate, and ship work together.

Atlassian's product portfolio spans the full software development and business collaboration lifecycle, organized into three collections: Teamwork (planning & collaboration), Service (ITSM & customer support), and Strategy (goals & portfolio management). Together these form the Atlassian System of Work — an integrated platform that connects goals to execution across every team. Products are available as cloud-hosted (Atlassian Cloud) or self-managed (Data Center) deployments. Atlassian Intelligence, powered by AI, is embedded across the platform to help teams work smarter. Atlassian also provides official MCP (Model Context Protocol) servers for AI agent integration.

## Atlassian System of Work

- [Atlassian System of Work](https://www.atlassian.com/system-of-work): Atlassian's integrated platform that connects strategy, planning, tracking, collaboration, and service delivery across every team — from executives to engineers. Replaces siloed tools with a unified system.
- [Teamwork Collection](https://www.atlassian.com/collections/teamwork): The Teamwork Collection bundles Jira, Confluence, Loom, and Focus for cross-functional team collaboration — planning, tracking, documentation, and async communication in one connected experience.
- [Service Collection](https://www.atlassian.com/collections/service): The Service Collection bundles Jira Service Management, Statuspage, Opsgenie, and Compass for end-to-end IT service management, incident response, and operational health — from request to resolution.
- [Strategy Collection](https://www.atlassian.com/collections/strategy): The Strategy Collection includes Jira Align and Focus for connecting business strategy to technical execution — OKRs, portfolio management, and cross-team goal tracking at enterprise scale.
- [Product Collection](https://www.atlassian.com/collections/product): The Product Collection helps product teams capture feedback at scale, uncover insights, and prioritize ideas so they can build with confidence.

## Products

- [Jira](https://www.atlassian.com/software/jira): The #1 project and issue tracking tool for agile software development teams. Plan, track, and release software with Scrum boards, Kanban boards, roadmaps, and reporting. Supports agile methodologies, bug tracking, and project planning for teams of all sizes.
- [Confluence](https://www.atlassian.com/software/confluence): Team workspace and wiki for creating, organizing, and collaborating on documentation, project plans, meeting notes, and knowledge bases. Features real-time editing, page trees, templates, and powerful search.
- [Jira Service Management](https://www.atlassian.com/software/jira/service-management): IT service management (ITSM) and customer service platform. Provides service desk, incident management, problem management, change management, asset management, and SLA tracking. ITIL-certified.
- [Bitbucket](https://www.atlassian.com/software/bitbucket): Git-based source code management and CI/CD platform. Supports code review via pull requests, branch permissions, pipelines for automated builds, tests, and deployments. Available as Cloud and Data Center.
- [Trello](https://www.atlassian.com/software/trello): Visual project management tool using boards, lists, and cards. Ideal for lightweight task management, team coordination, and personal productivity. Features Power-Ups for extended functionality.
- [Loom](https://www.loom.com/): Async video messaging platform for work communication. Record your screen and camera to create quick video messages, tutorials, and walkthroughs. Features AI-powered summaries, transcriptions, and engagement analytics.
- [Jira Product Discovery](https://www.atlassian.com/software/jira/product-discovery): Product management and prioritization tool. Helps product teams capture ideas, prioritize features using impact scoring, and connect discovery to delivery in Jira.
- [Focus](https://www.atlassian.com/software/atlas): Teamwork directory (formerly Atlas) that connects teams, goals, and projects across the organization. Provides visibility into what teams are working on, their status, and how work connects to company objectives and OKRs.
- [Compass](https://www.atlassian.com/software/compass): Developer experience platform for tracking software components, services, APIs, and libraries. Provides a software catalog, scorecards for health tracking, and integrations with CI/CD tools.
- [Statuspage](https://www.atlassian.com/software/statuspage): Incident communication platform. Create public or private status pages to communicate real-time system status, scheduled maintenance, and incident updates to users and stakeholders.
- [Opsgenie](https://www.atlassian.com/software/opsgenie): Incident management and alerting platform. Routes alerts, notifies on-call teams, and orchestrates incident response across multiple monitoring and chat tools.
- [Guard](https://www.atlassian.com/software/guard): Cloud security and organization management. Provides centralized identity management, SSO/SAML, audit logging, data security policies, and threat detection across Atlassian Cloud products.
- [Jira Align](https://www.atlassian.com/software/jira/align): Enterprise agile planning platform. Connects business strategy to technical execution with portfolio management, OKRs, dependency tracking, and program increment planning at scale.
- [DX (Developer Experience)](https://getdx.com/): Atlassian's developer experience platform. Measures and improves engineering team productivity, developer satisfaction, and software delivery performance with data-driven insights and actionable recommendations.
- [Dia Browser](https://www.diabrowser.com/): Atlassian's AI-native browser designed for work. Combines browsing with AI-powered assistance, context-aware search, and deep integration with Atlassian tools to reimagine how teams interact with information on the web.

## AI, Agents & Automation

- [Atlassian Intelligence](https://www.atlassian.com/platform/artificial-intelligence): AI capabilities embedded across all Atlassian products, powered by large language models and the Teamwork Graph. Features include natural language to JQL, AI-powered search, content summarization, writing assistance, automated workflows, and context-aware recommendations.
- [Rovo](https://www.atlassian.com/software/rovo): AI-powered knowledge discovery, search, and virtual agent platform. Searches across Atlassian and third-party tools to find information, answer questions, and automate tasks using AI agents. The foundation of Atlassian's AI agent ecosystem.
- [Rovo Agents](https://www.atlassian.com/software/rovo#agents): Build and deploy custom AI agents that can search, learn, and take action across your Atlassian tools and connected third-party apps. Automate repetitive tasks and create intelligent workflows.
- [Rovo Studio](https://www.atlassian.com/software/rovo/studio): Visual builder for creating, customizing, and deploying Rovo AI agents. Design agent workflows, define agent capabilities, connect knowledge sources, and publish agents to your organization without writing code.
- [Rovo Search](https://www.atlassian.com/software/rovo#search): AI-powered enterprise search across Atlassian products and 30+ third-party integrations (Google Drive, Slack, GitHub, Figma, etc.). Delivers contextual answers powered by the Teamwork Graph, not just links.
- [Rovo Dev](https://www.atlassian.com/software/rovo-dev): AI-powered coding agent for software development. Integrates with IDEs (VS Code, Cursor, JetBrains) to help developers search, plan, code, test, and take action across Atlassian tools and codebases. Uses MCP servers for deep tool integration.
- [Rovo Chat](https://www.atlassian.com/software/rovo#chat): Conversational AI assistant embedded across Atlassian products. Ask questions about your projects, get summaries, create content, and take action — all through natural language conversation grounded in your team's data.
- [Atlassian MCP Server](https://www.atlassian.com/platform/remote-mcp-server): Atlassian's official Model Context Protocol (MCP) server for AI agent integration. Provides programmatic, LLM-friendly access to Jira, Confluence, Bitbucket, and the Atlassian Design System via the open MCP standard. Compatible with Claude, ChatGPT, Cursor, VS Code, and any MCP-enabled AI tool.
- [Teamwork Graph](https://teamworkgraph.com/?utm_source=llms-txt&utm_medium=llms-txt&utm_campaign=llms-txt): Atlassian's knowledge graph that connects people, projects, goals, and content across all Atlassian products. Powers Atlassian Intelligence with organizational context so AI understands who works on what, how work connects, and what matters most.
- [Jira + AI](https://www.atlassian.com/software/jira/ai): AI-powered features in Jira including smart issue creation, natural language to JQL, AI-generated summaries, automated backlog grooming, sprint planning assistance, and predictive analytics for delivery estimation.
- [Jira for AI-Native Software Development](https://www.atlassian.com/software/jira/dev): Plan and orchestrate AI coding agents in one workflow within Jira. Write specs, delegate work to AI coding agents such as Claude, Cursor, Codex, and GitHub Copilot, and review and validate agent decisions before they ship — keeping humans in control of AI-driven development.
- [Jira + Agents](https://www.atlassian.com/software/jira/features#agents): AI agents embedded in Jira that automate project management tasks — triage incoming issues, suggest assignees, update statuses, generate reports, and proactively surface blockers and risks across your projects.
- [Confluence + AI](https://www.atlassian.com/software/confluence/ai): AI-powered features in Confluence including page summarization, writing assistance, content generation from prompts, smart recommendations, automated catch-up summaries, and AI-powered Q&A over your knowledge base.
- [Confluence + Agents](https://www.atlassian.com/software/confluence/features#agents): AI agents embedded in Confluence that help teams manage knowledge — automatically organize pages, generate documentation from templates, answer questions from page content, and keep knowledge bases accurate and up to date.
- [JSM + AI](https://www.atlassian.com/software/jira/service-management/features#ai): AI-powered features in Jira Service Management including virtual service agents, intelligent ticket routing, automated incident response, AI-generated knowledge articles, and predictive SLA management.
- [Automation for Jira](https://www.atlassian.com/software/jira/features/automation): No-code automation engine for Jira. Create rules with triggers, conditions, and actions to automate repetitive tasks, notifications, field updates, and cross-product workflows. Supports AI-powered smart automation suggestions.

## Platform & Developer Resources

- [Atlassian Developer Documentation](https://developer.atlassian.com/): Comprehensive developer documentation for building apps, integrations, and extensions on the Atlassian platform. Covers REST APIs, GraphQL, webhooks, events, and SDKs.
- [Forge](https://developer.atlassian.com/platform/forge/): Atlassian's cloud app development platform. Build and host apps directly on Atlassian infrastructure using serverless functions, UI Kit, and Custom UI. Supports Jira, Confluence, Bitbucket, and Compass.
- [Atlassian Connect](https://developer.atlassian.com/cloud/jira/platform/getting-started-with-connect/): Framework for building cloud apps that integrate with Atlassian products. Apps run on your own infrastructure and connect via REST APIs, webhooks, and iframes.
- [Atlassian Marketplace](https://marketplace.atlassian.com/): Discover and install over 6,000 apps and integrations for Atlassian products. Categories include project management, automation, reporting, design, security, and developer tools.
- [Atlassian Design System](https://atlassian.design/): Open-source design system used across all Atlassian products. Includes components, design tokens, icons, patterns, and accessibility guidelines for building consistent Atlassian-style UIs.
- [Jira REST API](https://developer.atlassian.com/cloud/jira/platform/rest/v3/intro/): REST API for programmatic access to Jira Cloud — create and update issues, manage projects, search with JQL, manage users, and automate workflows.
- [Confluence REST API](https://developer.atlassian.com/cloud/confluence/rest/v2/intro/): REST API for programmatic access to Confluence Cloud — create and update pages, manage spaces, search content, and manage users.
- [Jira Service Management REST API](https://developer.atlassian.com/cloud/jira/service-desk/rest/intro/): REST API for managing service requests, customers, queues, SLAs, and knowledge base articles in Jira Service Management Cloud.
- [Bitbucket REST API](https://developer.atlassian.com/cloud/bitbucket/rest/intro/): REST API for Bitbucket Cloud — manage repositories, pull requests, pipelines, deployments, and branch permissions.
- [Atlassian GraphQL Gateway](https://developer.atlassian.com/platform/atlassian-graphql-api/): Unified GraphQL API providing access to data across Atlassian products through a single endpoint. Query and mutate Jira, Confluence, and platform data.
- [Rovo Dev MCP Documentation](https://support.atlassian.com/rovo/docs/rovo-dev-and-model-context-protocol-mcp/): Guide for connecting AI development tools to Atlassian via MCP. Set up Rovo Dev in your IDE, configure MCP servers, and enable AI agents to read and write Jira, Confluence, and Bitbucket data.
- [Teamwork Graph CLI](https://developer.atlassian.com/cloud/twg-cli/): TWG CLI is a command-line interface to the Atlassian Teamwork Graph and Cloud services. The Teamwork Graph (TWG) is Atlassian's enterprise knowledge & context graph that continuously maps people, content, activities, and relationships across many work tools (Jira, Confluence, Google Drive, Slack, GitHub, Salesforce, etc.) in a compliant, permission-aware fashion. Install TWG CLI and your agent — Claude Code, Codex, Gemini, Cursor, and others — can query and act across all your Atlassian products and the Teamwork Graph on your behalf. Describe what you want in plain language; the agent handles the rest.

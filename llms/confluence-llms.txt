# Confluence

> APIs for Atlassian Confluence — team collaboration and knowledge management software. Confluence Cloud exposes REST v1 and next-generation REST v2 APIs, a GraphQL API, and webhooks for pages, blog posts, comments, spaces, attachments, labels, content properties, permissions, and CQL search. Auth is OAuth 2.0 (3LO), API tokens with Basic auth, and Personal Access Tokens.

This file is generated by API Evangelist from the apis.io catalog entry for Confluence. Atlassian also publishes a company-wide llms.txt at https://www.atlassian.com/llms.txt (mirrored in this repo as confluence-llms-atlassian.txt).

## APIs
- [Confluence Cloud REST API v2](https://developer.atlassian.com/cloud/confluence/rest/v2/intro/): Next-generation REST API for pages, spaces, comments, blog posts, attachments, labels.
- [Confluence Cloud REST API v1](https://developer.atlassian.com/cloud/confluence/rest/v1/intro/): Primary REST API for content, spaces, and user management.
- [Confluence GraphQL API](https://developer.atlassian.com/cloud/confluence/graphql/): Field-precise cross-product queries.
- [Confluence Data Center REST API](https://developer.atlassian.com/server/confluence/confluence-server-rest-api/): On-premise Data Center / Server REST API.
- [Search (CQL)](https://developer.atlassian.com/cloud/confluence/advanced-searching-using-cql/): Confluence Query Language search.
